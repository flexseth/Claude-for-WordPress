## Dynamic rendering of blocks
Has the content on this web page changed since the last time I was here?
If something has changed other than the post, there are dynamic blocks on this page.

## background
Dynamic blocks are blocks whose content is generated on the server each time the post is viewed. This allows the block to display content that can change based on external factors, such as the current date, user information, or data from an external API.
