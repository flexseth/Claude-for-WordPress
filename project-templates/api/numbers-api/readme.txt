Display interesting facts about numbers from the Numbers API.

== Description ==

The Numbers API block fetches and displays fascinating trivia, mathematical facts, dates, and year information about numbers. Perfect for educational content, fun facts sections, or adding engaging numerical content to your WordPress site.

= Features =

* Multiple Number Types: Choose between random numbers or specific numbers
* Four Fact Categories: Trivia, Math, Date, and Year facts
* Real-time API Integration: Fetches fresh facts from numbersapi.com
* WordPress Block Editor: Fully integrated with Gutenberg
* Dynamic Rendering: Server-side rendering with client-side API fetching
* Customizable Settings: Easy-to-use controls in the block inspector
* Internationalization Ready: Full i18n support
* Accessible: Follows WordPress accessibility standards

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/numbers-api` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the "Numbers API" block to any post or page through the block editor.

== Frequently Asked Questions ==

= What is the Numbers API? =

The Numbers API (numbersapi.com) is a free API that provides interesting facts about numbers, including trivia, mathematical properties, dates, and years.

= Do I need an API key? =

No, the Numbers API is free to use and does not require an API key.

= Can I use specific numbers? =

Yes! You can choose between random numbers or enter a specific number to get facts about.

= What types of facts are available? =

The block supports four types of facts:
* Trivia - Interesting facts about numbers
* Math - Mathematical properties and facts
* Date - Historical events on specific dates
* Year - Events from specific years

= Does this work with caching plugins? =

Yes, but note that the facts are fetched on the client side, so users will see fresh content even with caching enabled.
