## Claude POC Block for WordPress
This is a proof-of-concept WordPress plugin created using Claude Code. It demonstrates how to scaffold a custom Gutenberg block using the `@wordpress/create-block` package and follows WordPress coding standards and best practices.

## install
```
cd claude-poc-block
  npm start           # Development mode
  npm run build       # Production build
  npm run plugin-zip  # Create .zip for installation
```
